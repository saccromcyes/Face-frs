# FaceFenix — Face Recognition System using Deep Learning

FaceFenix is a real-time **Face Recognition System** built using **Deep Learning** and **Computer Vision** techniques.  
It uses `Facenet-PyTorch` for face embeddings, `FastAPI` as the backend, and `Streamlit` as the frontend to provide a full-stack facial recognition platform.

---

## Features

- **Face Detection & Recognition** using MTCNN and InceptionResnetV1 (VGGFace2)
- **Identity Registration** — Easily add new users with their image
- **Fast Recognition** — Instantly identify registered faces
- **Interactive Web UI** — Built using Streamlit
- **Deep Learning Powered** — Learns facial features with high accuracy
- **SQLite Database** — Stores embeddings, metadata, and user details
- **Cross-Origin Enabled** — Backend ready for frontend integration
